HackOlympia

Τι είναι το HackOlympia;

Καλωσορίσατε στην ιστοσελίδα του HackOlympia! Είμαστε μια κοινότητα μαθητών δευτεροβάθμιας εκπαίδευσης από όλη την Ελλάδα. Ο στόχος μας είναι η εκμάθηση προγραμματισμού μαζί. Μαθητές Γυμνασίου και Λυκείου από όλη την Ελλάδα, μπορούν να συμμετάσχουν την κοινότητά μας.

Τι κάνετε σε αυτήν την κοινότητα;

Η κοινότητά μας λειτουργεί στο Discord. Κάθε Σάββατο από τις 4:00 μ.μ. μέχρι τις 5:30 μ.μ., είμαστε όλοι σε μια κλήση στο Zoom και κάνουμε:

  • Μαθήματα
  • Kahoot!
  • Ομαδικά/Ατομικά Project
  • Και άλλα

Κάνουμε μαθήματα σχετικά με:

  • Την κατασκευή ιστοσελίδων
  • Βασικές έννοιες προγραμματισμού
  • Ρομποτική
  • Κατασκευή εφαρμογών

Μπορώ να συμμετάσχω και εγώ;

Η κοινότητά μας θέλει να δέχεται όσο δυνατόν, περισσότερους μαθητές. Γι'αυτό δεν έχουμε πολλές απαιτήσεις. Πρέπει μόνο:

  • Να είσαι μαθητής Α' Γυμνασίου έως Γ'Λυκείου. Δυστυχώς δεν δεχόμαστε μέλη Δημοτικού ή Πανεπιστημίου
  • Να θέλεις να μάθεις! Δεν χρειάζεται να έχεις γνώσεις προγραμματισμού! Έχουμε συσκέψεις για νέα μέλη χωρίς γνώσεις προγραμματισμού.

Τέλεια! Που υπογράφω;

Για να κάνεις εγγραφή, θα πρέπει να κάνεις 2 πράγματα.

  1. Να μπεις στον server μας στο Discord.
  2. Να συμπληρώσεις την φόρμα εγγραφής, με την οποία θα πάρουμε περισσότερες πληροφορίες για εσένα.
