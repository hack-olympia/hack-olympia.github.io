HackOlympia

Τι είναι το HackOlympia;

Καλωσορίσατε στην ιστοσελίδα του HackOlympia! Είμαστε μια κοινότητα μαθητών δευτεροβάθμιας εκπαίδευσης από όλη την Ελλάδα. Ο στόχος μας είναι η εκμάθηση προγραμματισμού μαζί. Μαθητές Γυμνασίου και Λυκείου από όλη την Ελλάδα, μπορούν να συμμετάσχουν την κοινότητά μας.

Τι κάνετε σε αυτήν την κοινότητα;

Η κοινότητά μας λειτουργεί στο Discord. Κάθε Σάββατο από τις 7:00 μ.μ. μέχρι τις 8:30 μ.μ., είμαστε όλοι σε μια κλήση και κάνουμε:

  • Μαθήματα
  • Kahoot!
  • Ομαδικά/Ατομικά Project
  • Και άλλα

Κάνουμε μαθήματα σχετικά με:

  • Την κατασκευή ιστοσελίδων
  • Βασικές έννοιες προγραμματισμού
  • Ρομποτική
  • Κατασκευή εφαρμογών

Μπορώ να μπω και εγώ;

Η κοινότητά μας θέλει να δέχεται όσο δυνατόν περισσότερα άτομα. Γι'αυτό δεν έχουμε πολλές απαιτήσεις. Πρέπει μόνο:

  • Να είσαι μαθητής Α' Γυμνασίου έως Γ'Λυκείου.
  • Να θέλεις να μάθεις. Δε χρειάζεται να έχεις γνώσεις προγραμματισμού! Έχουμε συναντήσεις για νέα μέλη χωρίς γνώσεις προγραμματισμού.

Τέλεια! Πού υπογράφω;

Για να κάνεις εγγραφή, θα πρέπει να κάνεις 2 πράγματα.

  1. Να μπεις στον server μας στο Discord.
  2. Να συμπληρώσεις τη φόρμα εγγραφής, με την οποία θα πάρουμε περισσότερες πληροφορίες για εσένα.
